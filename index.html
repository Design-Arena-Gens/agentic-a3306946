Setup Guide (Free, No Paid Tools Required)

  1. Install n8n (free): docker run -it --rm -p 5678:5678 n8nio/n8n or use a free VM.
  2. Import JSON: In n8n, go to Workflows ? Import from File and choose the downloaded JSON.
  3. Credentials: Fill placeholders (Google Sheets, Gmail, Telegram, Webhook URLs). All optional AI nodes are disabled by default.
  4. Enable: Toggle each workflow Active. Use the Master Orchestrator to schedule everything via Cron.
  5. Customize: Edit the Set/Function nodes labeled "Config" to add your product feeds, topics, and templates.

Tip: Start with the Master Orchestrator, then gradually enable vertical workflows. Every workflow includes clear labels so you can extend without coding.

FAQ

Do these require paid AI keys?

No. All AI steps are optional and disabled by default. The templates work with rules/templates. If you add an AI provider, enable the AI node.

Where do outputs go?

By default: Google Sheets/CSV + optional notifications (Telegram/Email). Configure inside each workflow.

Can I run this on free infra?

Yes. Self-host n8n on a free-tier VM or local machine with Cron. No paid tools required.
